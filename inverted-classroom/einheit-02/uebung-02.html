CSS 2.0 Selektoren

E matches any E Element.

Beispiel: em in blau.

E F matches any F Element that is a descendent of an E Element

Beispiel:
  • Funktioniert über Tag Ebenen hinweg (z.B. von einem <ul> ausgehend) (solange als Parent irgendwann ein <ul> vorkommt).

E > F matches any F Element that is a child of E

Dies sollte eingefärbt werden. Dies nicht (... Bzw. über den ersten Sektor in blau).
